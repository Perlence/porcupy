Library reference
=================

Game objects
------------

Porcupy provides access to many built-in objects to interact with the game.

Yozhik
``````

Built-in list ``yozhiks`` consist of 10 *Yozhik* instances.
Each instance has the following properties:

**frags**

   Number of frags.

**pos_x**, **pos_y**

   Position of yozhik.
